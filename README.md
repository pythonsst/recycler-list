# react-native-recyclerlist

RecyclerList is a Fabric-first, native-driven virtualized list for React Native.

## Installation


```sh
npm install react-native-recyclerlist react-native-nitro-modules

> `react-native-nitro-modules` is required as this library relies on [Nitro Modules](https://nitro.margelo.com/).
```


## Usage


```js
import { RecyclerlistView } from "react-native-recyclerlist";

// ...

<RecyclerlistView color="tomato" />
```
